# International Day - Bangladesh Heritage Showcase

This is a Next.js application built to showcase the rich culture, heritage, and culinary traditions of Bangladesh. It was created for "International Day" to provide an interactive and engaging experience for users to explore facts, historical landmarks, and traditional recipes.

## Features

- **Interactive Spin Wheel:** A fun way to discover random facts about Bangladesh.
- **Heritage Gallery:** A curated collection of facts about Bangladesh's history, nature, and GI (Geographical Indication) products.
- **Bengali Kitchen:** A showcase of traditional Bangladeshi recipes.
- **Modern UI:** A clean and elegant design with a "glass effect" for a modern look and feel.
- **Responsive Design:** The application is designed to work on both desktop and mobile devices.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animation:** [Framer Motion](https://www.framer.com/motion/)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18.x or later)
- npm, yarn, or pnpm

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/mehedihassandev/International-Day.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Run the development server
    ```sh
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
